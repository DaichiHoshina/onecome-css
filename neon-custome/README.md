# わんこめネオンテンプレート - カスタム版

OneCommeのネオンテンプレートをカスタマイズした軽量版です。

---

## 📝 カスタム内容

### 1. 各コメントを独立したカード表示
- 各コメントが個別のボックスとして表示
- コメント間に余白を追加
- 角丸のカード形式

### 2. ネオン光彩の軽量化
- **8層 → 3層**に削減（GPU負荷 62%削減）
- ぼかし半径を縮小（32px → 9px）
- スーパーチャット等の光彩も同様に軽量化

### 3. 可読性の向上
- テキスト色: 黒（背景とのコントラスト向上）
- 背景色: 薄いグレー（#e0e0e0）

---

## 🚀 使用方法

### OBS Studioで使用
1. **Browser Source** を追加
2. **ローカルファイル** を選択: `index.html`
3. **幅**: 1920、**高さ**: 1080（お好みで調整）
4. **完了**

---

## ⚙️ カスタマイズ

### 背景色を変更
`style.css` の `.comment` を編集：
```css
background-color: #e0e0e0 !important;  /* お好みの色に */
```

### コメント間の余白を調整
```css
margin-bottom: 16px !important;  /* 12px → 16px で広く */
```

### ネオン強度を調整
`style.css` の `:root` を編集：
```css
--lcv-neon-shadow: 0 0 3px #fff, 0 0 6px var(--neon-color), 0 0 9px var(--neon-color);
```
- 強くしたい: `9px` → `12px`
- 弱くしたい: `9px` → `6px`
